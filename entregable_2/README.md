# 📚 Entregable Taller 2: Revisión de Literatura

Este directorio contiene los artefactos y documentos correspondientes al **Taller 2 de Pensamiento Crítico**.

## 📌 Contenido

1. **`Taller2_Revision_Literatura.tex`**: Código fuente en LaTeX que consolida el entregable formal.
2. Este documento cubre la estructuración de la revisión sistemática de literatura orientada a modelos probabilísticos en esports, incluyendo:
   - Mini-Protocolo (PICOC, Pregunta de Investigación, Cadena de Búsqueda).
   - Tabla con Criterios de Inclusión y Exclusión.
   - Evaluación de Calidad (QA Checklist) aplicada a un artículo representativo.
   - Matriz de extracción para 5 artículos del estado del arte.
   - Análisis crítico de patrones, sesgos de la literatura actual, vacíos y su implicación directa para este proyecto de CS:GO con estimación bayesiana.

Todo el texto está listo para compilar en Overleaf o cualquier distribución de LaTeX.

## 📊 Resumen del proyecto
Modelo de Regresión Logística Bayesiana Jerárquica para predecir victorias en CS:GO profesional, usando métricas de HLTV.org. Variables clave: Diferencial de Rating e Impacto. Método de inferencia: MCMC (PyMC).

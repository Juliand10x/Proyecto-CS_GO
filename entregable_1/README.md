# 🔍 Entregable Taller 1: Análisis Crítico

En esta carpeta se desarrollan los artefactos correspondientes al **Taller 1** del proyecto de investigación.

## 📌 Contenido

1. **`Evaluacion_Critica_Esports.md`**: Versión extendida en Markdown del Marco de Análisis Crítico aplicado a la investigación de Agrawal et al. sobre *Counter-Strike: Global Offensive*.
2. **`Evaluacion_Critica_Esports.tex`**: Código fuente en LaTeX exacto al Markdown, listo para compilarse y entregarse como documento PDF formal.

Este análisis aborda de manera profunda la Ingeniería de Datos y de Software aplicada a los *demofiles* de CS:GO, criticando sus supuestos, métodos y generalizaciones causales para fundamentar la necesidad de nuestro propio enfoque probabilístico y predictivo (Regresión Logística Bayesiana).
